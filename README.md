# Instructions
1. Make sure all prerequisites are installed<br>
2. Run ```cellular automata.py```<br>
3. Select an image<br>
4. Watch your Conway's Game of Life simulation!<br>
5. You can pause to change cells as well as modify the simulation speed using the slider, have fun!
